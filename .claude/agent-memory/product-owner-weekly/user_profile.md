**Hard constraint**: every external service must stay on free tier. This is non-negotiable — do not propose features that would require a paid Gemini or Firebase plan.

**Timezone default**: Asia/Ho_Chi_Minh (Vietnam). Scheduled features default to this timezone.

**Collaboration style**: the developer reads the code carefully. Proposals should reference exact file names and function signatures, not abstract descriptions. Cost math must be quantified, not hand-wavy.
