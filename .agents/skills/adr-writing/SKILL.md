# ADRWriting

## Purpose

Document architectural decisions and tradeoffs.

## When to Use

Any significant technical choice.

## Instructions

- State context, options, decision, consequences, rollback plan, success metric.

## Heuristics

- Keep short; link to related tickets.

## Examples

ADR for switching to async messaging.
